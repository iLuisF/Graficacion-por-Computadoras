El modelo se descargo de 3D: 
http://graphics.stanford.edu/data/3Dscanrep/

El cual corresponde a bunny.tar.gz este(se adjuntó), el original se redimensiono ya que era demasiado pequeño.
Además para poder observar bien el modelo con el programa se pueden usar las configuraciones
con las que se cuenta por medio del teclado.

El archivo knots.c se modifico el indice en la funcion void GargarPoligonos(char *nombre)
al momento de cargar los vertices.

El archivo model.DAT contiene el modelo con el formato especifico para poder visualizarlo.
